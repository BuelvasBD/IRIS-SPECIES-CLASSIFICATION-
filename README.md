# Iris Species Dashboard

Interactive Machine Learning dashboard built with Streamlit for Iris flower classification using the Random Forest algorithm.

---

# Project Purpose

The purpose of this project is to demonstrate:

- Data visualization techniques
- Interactive dashboard development
- Machine Learning classification
- Model evaluation metrics
- User interaction with predictive models

The application allows users to explore the Iris dataset, visualize flower distributions, analyze feature importance, and predict flower species based on user input.

---

# Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Streamlit Option Menu

---

# Dataset

This project uses the famous Iris dataset from Scikit-learn.

The dataset contains measurements of iris flowers including:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target classes:

- Setosa
- Versicolor
- Virginica

---

# Machine Learning Model

The application uses a Random Forest Classifier for species prediction.

## Model Workflow

1. Load dataset
2. Split training and testing data
3. Train Random Forest model
4. Evaluate model performance
5. Generate predictions from user input
